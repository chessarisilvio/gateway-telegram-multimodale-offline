# Gateway Telegram Multimodale Offline

**Categoria**: progetto
**Difficoltà**: Facile
**Stima**: 3-5gg

## Descrizione

Estensione di openclaw in Node.js che integra moondream 1.4B e Whisper large-v3 per processare media in arrivo su Telegram. Pipeline: ricezione → frame extraction → descrizione visiva → trascrizione → routing a Qwen3.6 per risposta contestuale, tutto in locale.
